## CONTACTS MANAGER

Sistema simples de gerenciamento de contatos desenvolvido em Laravel.

#### FUNCIONALIDADES

- Cadastro de contatos
- Listagem
- Edição
- Exclusão

#### TECNOLOGIAS

- PHP 8+
- Laravel
- MySQL
- Blade

#### INSTALAÇÃO

1. Clone o repositório
   
   ``git clone https://github.com/viniciusma7/alfasoft.git``

2. Acesse a pasta do projeto

   ``cd alfasoft``

3. Instale as dependências

   ``composer install``

   ``npm install && npm run build``

4. Crie o arquivo de ambiente

   ``cp .env.example .env``

5. Gere a chave da aplicação

   ``php artisan key:generate``

6. Configure o banco de dados no arquivo .env

7. Rode as migrations

   ``php artisan migrate``

8. Rode a aplicação

   ``php artisan serve``

REQUISITOS

- PHP 8+
- Composer
- MySQL

OBSERVAÇÕES

Projeto desenvolvido para aprendizado em Laravel e prática de conceitos de backend.
